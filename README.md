Este projeto consiste no desenvolvimento de um e-commerce de floricultura chamado Floricultura Margarida. A ideia do site surgiu com o objetivo de ir além de uma loja virtual comum, trazendo também um espaço de interação entre os usuários, funcionando como uma espécie de comunidade ou blog sobre plantas, flores e jardinagem. Dessa forma, o projeto une a venda de produtos com a troca de experiências e curiosidades entre os clientes.

O principal objetivo do sistema é simular o funcionamento de um e-commerce real, oferecendo uma navegação simples, visual agradável e fácil compreensão para o usuário. O site permite visualizar produtos, adicionar itens ao carrinho, alterar quantidades, remover produtos e acompanhar o valor total da compra. Além disso, o projeto conta com um diferencial, que é a área de curiosidades, onde os usuários podem curtir conteúdos, enviar comentários e receber um cupom de desconto como forma de incentivo à participação.

Para o desenvolvimento do projeto foram utilizadas tecnologias voltadas ao front-end, sendo elas HTML5 para a estrutura das páginas, CSS3 para a estilização e criação do layout responsivo, e JavaScript para a implementação das funcionalidades e interações do site. Também foram utilizados Bootstrap para auxiliar na responsividade, Font Awesome para os ícones e jQuery em algumas partes do projeto. O armazenamento das informações é feito por meio do LocalStorage do navegador, permitindo que dados como carrinho de compras, curtidas, comentários e cupons permaneçam salvos mesmo após atualizar a página.

A estrutura do projeto é organizada em arquivos HTML responsáveis pelas diferentes páginas do site, como a página inicial, a página de produtos, a página de curiosidades e a página do carrinho de compras. Além disso, o projeto possui pastas específicas para armazenar imagens utilizadas no site, facilitando a organização e manutenção dos arquivos. Toda essa estrutura está reunida em um repositório no GitHub, junto com este arquivo README, que contém as informações sobre o projeto e sua execução.

Entre as principais funcionalidades implementadas estão o carrinho de compras com cálculo automático de subtotal e total, a aplicação de cupom de desconto, o sistema de curtidas “Amei”, o envio de comentários pelos usuários e a liberação de um cupom promocional após a interação com a comunidade. O site também foi desenvolvido de forma responsiva, permitindo seu uso em diferentes dispositivos, como computadores, tablets e celulares.

Para executar o projeto, não é necessário instalar nenhum programa ou dependência adicional. Basta acessar o repositório, abrir o arquivo index.html em um navegador web ou acessar o projeto diretamente pelo GitHub Pages. Todo o funcionamento acontece no próprio navegador, sem necessidade de servidor ou banco de dados.

O projeto Floricultura Margarida foi desenvolvido como Trabalho de Conclusão de Curso, com o objetivo de colocar em prática os conhecimentos adquiridos ao longo do curso, especialmente nas áreas de desenvolvimento web, organização de código, design responsivo e lógica de programação. O resultado final é um site funcional, organizado e interativo, que representa de forma clara a aplicação dos conteúdos estudados.
